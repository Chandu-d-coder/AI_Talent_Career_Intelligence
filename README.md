# 🚀 TalentLens AI

### Workforce & Career Intelligence Platform

TalentLens AI is a portfolio-grade workforce intelligence project designed to transform large-scale AI workforce data into meaningful career, hiring, salary, skill, and workforce insights.

Most job market projects stop at basic charts and descriptive statistics. This project takes a different approach by treating workforce data as a strategic asset and using it to answer real-world career, hiring, and workforce intelligence questions.

The goal is not just to analyze data, but to think like a Workforce Intelligence Analyst, Data Analyst, and future AI Engineer.

## 🎯 Problem Statement

Organizations, recruiters, and aspiring AI professionals often struggle to answer questions such as:

* Which AI specializations are growing fastest?
* Which AI skills are currently most in demand?
* Which AI roles offer the highest compensation?
* Which countries provide the strongest workforce opportunities?
* What factors influence career growth?
* Which careers appear most resilient to automation?
* How does workforce demand vary across AI domains?

TalentLens AI was created to explore these questions using workforce intelligence techniques and data-driven analysis.

## 💻 Local Setup

Clone the repository:

```bash
git clone https://github.com/Chandu-d-coder/AI_Talent_Career_Intelligence.git
```

Move into the project directory:

```bash
cd AI_Talent_Career_Intelligence
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```
