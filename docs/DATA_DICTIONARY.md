# Data Dictionary

## Purpose

This document provides definitions for all dataset attributes used throughout the TalentLens AI project.

| Column Name             | Data Type   | Description                   |
| ----------------------- | ----------- | ----------------------------- |
| job_role                | Categorical | AI Job Role                   |
| ai_specialization       | Categorical | AI Domain Specialization      |
| experience_level        | Categorical | Experience Category           |
| experience_years        | Numeric     | Years Of Experience           |
| education_required      | Categorical | Required Education Level      |
| salary_usd              | Numeric     | Annual Salary In USD          |
| bonus_usd               | Numeric     | Annual Bonus In USD           |
| salary_percentile       | Numeric     | Salary Ranking Percentile     |
| job_openings            | Numeric     | Number Of Open Positions      |
| skill_demand_score      | Numeric     | Market Demand For Skills      |
| hiring_difficulty_score | Numeric     | Hiring Complexity Indicator   |
| interview_rounds        | Numeric     | Average Interview Stages      |
| offer_acceptance_rate   | Numeric     | Offer Acceptance Percentage   |
| layoff_risk             | Numeric     | Layoff Risk Indicator         |
| automation_risk         | Numeric     | Automation Exposure Indicator |
| job_security_score      | Numeric     | Employment Stability Score    |
| company_size            | Categorical | Company Size Category         |
| company_rating          | Numeric     | Company Rating Score          |
| company_funding_billion | Numeric     | Funding Value In Billions     |
| ai_adoption_score       | Numeric     | AI Adoption Maturity Score    |
| ai_maturity_years       | Numeric     | Years Of AI Maturity          |
| employee_satisfaction   | Numeric     | Employee Satisfaction Score   |
| work_life_balance_score | Numeric     | Work Life Balance Rating      |
| vacation_days           | Numeric     | Annual Vacation Days          |
| weekly_hours            | Numeric     | Weekly Working Hours          |
| country                 | Categorical | Employee Country              |
| economic_index          | Numeric     | Economic Indicator            |
| cost_of_living_index    | Numeric     | Cost Of Living Metric         |
| tax_rate_percent        | Numeric     | Tax Percentage                |
| career_growth_score     | Numeric     | Career Growth Potential       |
| promotion_speed         | Numeric     | Promotion Velocity Indicator  |
